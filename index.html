My Platformer Game

The name of this game is called Ninja Adventure. It is a 2D cartoon platformer where a player controls a ninja, collects coins, defeats enemies, and completes timed levels with increasing difficulty. It was developed for my Game Development unit in my BTEC course.

Controls:

  • Left / Right Arrow Keys - Move
  • Shift + Left / Right Arrow Key - Slide
  • Space Key - Jump
  • X Key - Attack
  • F Key - Throw knife
